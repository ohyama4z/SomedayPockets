# ADR-002: 開発環境設定

- **日付**: 2026-03-14
- **ステータス**: 承認済み
- **チケット**: #15

## コンテキスト

プロジェクト初期セットアップにあたり、開発環境のツール類を決定する必要がある。

## 決定

### パッケージマネージャー
- **npm**

### ローカル開発実行
- **Wrangler**（ローカルモード）で Cloudflare Pages / Workers をローカル実行
- **D1** もWranglerのローカルモードで動作確認

### テスト
- ユニットテスト: **Vitest**
- E2Eテスト: **Playwright**

## 理由

- **npm**: 特別な要件がなく馴染みがあるため。pnpmはディスク効率・速度の利点があるが、個人開発では差が小さい
- **Wrangler ローカルモード**: Cloudflare環境を本番に近い形でローカル再現できる。D1もローカルモードで十分
- **Vitest**: ESMネイティブで高速、Next.js/TypeScript構成との相性が良い。Jestより設定が軽い
- **Playwright**: Next.js公式サポートあり、並列実行・複数ブラウザ対応。E2EのデファクトスタンダードはPlaywright
