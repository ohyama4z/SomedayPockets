# MyQuest - Claude Code プロジェクト設定

## プロジェクト概要
「やりたいことリスト」管理Webアプリケーション。個人開発。

## ディレクトリ構成
- `docs/` - プロダクトドキュメント（要件定義、設計、未決事項など）
- `app/` - アプリケーションコード
- `.claude/` - Claude Code設定

## コミュニケーション
- すべて日本語で行う（会話、コード内コメント、コミットメッセージ含む）
- 変数名・関数名などの識別子は英語

## ドキュメント運用
- 要件や設計の決定事項は `docs/` 配下に残す
- 未決事項は `docs/open-questions.md` で管理し、解決したら `requirements.md` に反映する
- Claude Codeのメモリ (`~/.claude/projects/.../memory/`) にはプロジェクト横断の学びや安定したパターンを記録する

## 改善提案
- 運用・開発プロセス・Claude Codeの活用方法について、気づいた改善点はその都度積極的に提案する
